## Topics Covered

### Core Framework (32 topics)

#### 1. Purpose and Objectives
- Why valuation rules are needed
- Preventing arbitrary customs values
- Protecting tariff concessions
- Fair, uniform, and neutral system

#### 2. Historical Development
- **GATT 1947** - Original valuation provisions
- **GATT Framework** - Articles V, VII, VIII, IX, X
- **Tokyo Round** - 1979 Protocol adoption
- **Uruguay Round** - 1994 agreement creation
- **Current Rules** - Modern framework

#### 3. Agreement Structure
- **Structure** - Four parts overview
- **General Principles** - Core principles in preamble
- **Part I Contents** - Articles 1-17
- **Annexes** - Annexes I, II, III
- **Final Provisions** - Articles 21-24

#### 4. Administration
- **Administration** - WTO and WCO committees
- **Special Treatment** - Article 20 for developing countries
- **WTO Accession** - Requirements for new members
- **Notifications** - Transparency obligations

#### 5. Valuation Methods (6 methods)
- **Valuation Methods** - Six authorized methods overview
- **Method 1** - Transaction value (primary method)
- **Alternative Methods** - Methods 2-6 hierarchy
- **Fall-back Method** - Method 6 application

#### 6. Transaction Value Details
- **Method 1** - Price actually paid or payable
- **Article 8** - Adjustments to transaction value
- **Incidental Exclusions** - What can be excluded
- **Method 1 Rejection** - When transaction value cannot be used
- **Method 1 Failure** - Reasons for rejection

#### 7. Prohibited Practices
- **Prohibited Methods** - Seven prohibited valuation bases
  - Domestic market prices
  - Minimum values
  - Arbitrary/fictitious values
  - Higher of two values
  - Export prices to third countries
  - Cost of production (except computed value)
  - Selling price of domestic goods

#### 8. Related Party Transactions
- **Related Parties** - Definition and criteria
- **Circumstances of Sale** - Examination methods
- **Test Values** - Measuring against comparable transactions

#### 9. Procedural Safeguards
- **Reasonable Doubts** - Customs procedures
- **Importer Protections** - Rights and appeals
- **General Objective** - Balance in customs operations

#### 10. Summary
- **Recap** - Transaction value formula

## Key Concepts to Master

### Transaction Value Formula
Transaction Value = Price Actually Paid or Payable + Article 8 Adjustments

### Article 8 Adjustments (if not already included)
1. Commissions and brokerage (except buying commissions)
2. Containers and packaging
3. Assists (materials, tools, engineering)
4. Royalties and licence fees
5. Proceeds of subsequent resale
6. International transport costs

### Related Party Criteria (8 types)
1. Family members
2. Business partners
3. Employer-employee
4. Officers/directors
5. 5%+ voting stock ownership
6. Direct/indirect control
7. Both controlled by third person
8. Together control third person

### Prohibited Methods (7 types)
1. Selling price of domestic goods
2. Higher of two alternative values
3. Domestic market price in exporting country
4. Cost of production (except computed value)
5. Export price to third country
6. Minimum customs values
7. Arbitrary or fictitious values

### Sequential Valuation Methods
1. Transaction value
2. Transaction value of identical goods
3. Transaction value of similar goods
4. Deductive value
5. Computed value
6. Fall-back method
