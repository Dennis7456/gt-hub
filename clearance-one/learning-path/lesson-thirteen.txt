Topic 12: Tax Assessment, CRSP, and Warehouse Rent Calculation

Topic goal: By the end of this topic, the learner should be able to:
- compute Customs value using CIF,
- calculate import duty, excise duty, and VAT in the correct order,
- allocate value across multiple items using the factor method,
- explain and apply the CRSP method for used motor vehicles,
- calculate and interpret Customs warehouse rent basics.

Step 1: Learn
Purpose: introduce the tax logic and formulas simply.
Lesson format: 6 learning cards + 4 micro-checks
Estimated time: 5–7 minutes
Pass type: completion-based
XP: up to 100

Learning cards

Card 1: Customs value basics
- Customs value is the CIF amount in local currency
- CIF = FOB + Insurance + Freight, then multiplied by exchange rate

Card 2: Tax order for normal imports
- Step 1: compute Customs value (CIF)
- Step 2: compute import duty
- Step 3: compute excise duty
- Step 4: compute VAT
- total taxes payable = sum of all applicable taxes

Card 3: Basic formulas
- CIF in local currency = (FOB + Insurance + Freight) × exchange rate
- Import Duty = rate × CIF
- Excise Duty = rate × (CIF + Import Duty)
- VAT = rate × (CIF + Import Duty + Excise Duty)

Card 4: Multiple-item consignment
- if insurance and freight are shared, use the factor method
- factor = total CIF in local currency ÷ total FOB in foreign currency
- Customs value per item = item FOB × factor

Card 5: CRSP method for used motor vehicles
- CRSP = current retail selling price from manufacturer/dealer sources
- subtract depreciation and loaded taxes to derive Customs value
- if transaction value is higher than CRSP-derived value, use the higher value

Card 6: Warehouse rent
- 21 days are working days, not calendar days
- Customs Warehouse Date = 22nd working day from discharge
- warehouse rent formula: CBM × number of rent days × rate per CBM × exchange rate
- simplified note formula: $0.3 × exchange rate × CBM × number of days

Micro-check examples
- What does CIF stand for?
- What tax comes after import duty?
- What is the factor method used for?
- On which day does warehouse status start?

Pass rule
Complete all cards and answer at least 3 out of 4 correctly.

Bonus mini-section inside Topic 12: Worked examples from the notes
This should appear in review mode because it is very exam-friendly.

Example 1: Whiskey tax calculation
From the notes:
- FOB = USD 5000
- Insurance = USD 200
- Freight = USD 900
- Exchange rate = 130
- Import duty = 25%
- Excise = 65%
- VAT = 16%

Flow
- CIF = (5000 + 200 + 900) × 130 = 793,000
- Import Duty = 25% × 793,000 = 198,250
- Excise Duty = 65% × (793,000 + 198,250)
- VAT = 16% × (793,000 + 198,250 + Excise Duty)
- Total payable = sum of all taxes

Example 2: Multiple-item factor method
From the notes:
- Hammers FOB = 1000
- Helmets FOB = 800
- Freight = 500
- Insurance = 400
- Exchange rate = 100

Flow
- Total CIF = (1000 + 800 + 500 + 400) × 100 = 270,000
- Total FOB = 1800
- Factor = 270,000 ÷ 1800 = 150
- Hammers Customs value = 1000 × 150
- Helmets Customs value = 800 × 150

Example 3: CRSP method
The notes show:
- start from manufacturer/dealer retail value
- subtract depreciation
- reverse out taxes to reach Customs value
- compare with transaction value and choose the higher where required

These examples are ideal for practice drills and exam revision.

Key facts this topic should reinforce
By the end, the learner should know:
- CIF = FOB + Insurance + Freight
- Customs value in import-tax examples is CIF in local currency
- tax order is:
  * CIF
  * Import Duty
  * Excise Duty
  * VAT
- import duty base = CIF
- excise duty base = CIF + Import Duty
- VAT base = CIF + Import Duty + Excise Duty
- factor method is used for multiple-item consignments with shared freight/insurance
- CRSP = current retail selling price method for used motor vehicles
- where transaction value is higher than CRSP-derived value, use the higher value
- Customs warehouse date = 22nd working day from discharge
- warehouse-rent formula depends on:
  * CBM
  * number of rent days
  * rate per CBM
  * exchange rate
