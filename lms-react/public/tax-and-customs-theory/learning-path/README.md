## Features

### Gamification
- **XP System**: Earn 10-30 XP per lesson
- **Hearts System**: Start with 5 hearts, lose 1 per wrong answer
- **Streak Tracking**: Daily activity tracking with flame icon
- **Progress Tracking**: Visual progress bars and completion dots

- **Retry System**: Wrong answers are queued for retry until mastered
- **Spaced Repetition**: Review weak lessons automatically
- **Progressive Unlocking**: Complete previous units to unlock next
