# Quick User Flow Summary

## 🎮 Tax and Customs Theory Learning Path

### One-Minute Overview

**What it is:** A Duolingo-style gamified learning experience for customs education

**Structure:** 12 Units → 67 Lessons → 6 Lesson Types

**Goal:** Master customs concepts from basics to smart customs

---

## 📍 User Journey in 10 Steps

```
1. LAND ON MAP
   ↓ See 12 units, only Unit 1 unlocked
   
2. CLICK UNIT 1
   ↓ See 5 lessons in vertical path
   
3. START LESSON 1 (Learn)
   ↓ Read 5 cards + answer 3 micro-checks
   ↓ Earn 100 XP
   
4. COMPLETE LESSON 2 (Quiz)
   ↓ Answer 10 questions, need 60% to pass
   ↓ Earn 100 XP
   
5. COMPLETE LESSON 3 (Match)
   ↓ Match 8 pairs, need 70% to pass
   ↓ Earn 100 XP
   
6. COMPLETE LESSON 4 (Scenario)
   ↓ Solve 6 real-world scenarios
   ↓ Earn 100 XP
   
7. COMPLETE LESSON 5 (Checkpoint)
   ↓ Pass 12 hard questions at 75%
   ↓ Earn 100 XP + 50 XP mastery + 250 XP unit bonus
   
8. UNIT 1 COMPLETE! 🎉
   ↓ Unit 2 unlocks
   ↓ Progress: 8.3% (1/12 units)
   
9. REPEAT FOR UNITS 2-11
   ↓ Each unit builds on previous knowledge
   ↓ Progressive difficulty increase
   
10. COMPLETE UNIT 12 (Final Mastery)
    ↓ Final boss challenge
    ↓ COURSE COMPLETE! 🎓
    ↓ Total: 7,200+ XP, 67 lessons mastered
```

---

## 🎯 6 Lesson Types Explained

| Type | Icon | What User Does | Pass % | Questions |
|------|------|----------------|--------|-----------|
| **Learn** | 📚 | Read 5 cards + 3 micro-checks | N/A | 8 interactions |
| **Quiz** | 🧠 | Answer multiple choice | 60% | 10 questions |

## 💎 Gamification Features

### Hearts System ❤️
- Start with 5 hearts
- Lose 1 per wrong answer
- Recover: wait 1 hour or practice old lessons

### XP System ⭐
- 100 XP per lesson
- 250 XP per unit completion
- 50 XP mastery bonuses
- Total: 7,200+ XP possible

### Streak System 🔥
- +1 day for daily activity
- +20 XP bonus every 5 days
- 2 streak freezes available

### Progress Tracking 📊
- Visual map shows all units
- Completion percentage
- Mastery levels: Introduced → Practicing → Strong → Mastered

---

## 🏆 Completion Criteria

✅ Complete all 67 lessons  
✅ Pass all 12 checkpoints (75%+)  
✅ Earn 7,200+ XP  
✅ Master customs concepts

## 💡 Key User Actions

**To Progress:**
1. Click unlocked lesson
2. Complete the activity
3. Pass the threshold
4. Earn XP
5. Unlock next lesson
6. Repeat until unit complete
7. Move to next unit

**If Failed:**
- Review material
- Click "Try Again"
- Retry lesson (no penalty)
- Pass to continue

**After Completion:**
- Practice mode unlocked
- Can replay any lesson
- Review for exams
- Track improvement
