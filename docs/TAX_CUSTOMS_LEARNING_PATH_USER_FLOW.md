# Tax and Customs Theory Learning Path - User Flow

## Overview

The Tax and Customs Theory learning path is a **Duolingo-style gamified learning experience** with 12 units, 67 lessons, and a progressive difficulty system. Users learn about customs administration from basic definitions to advanced smart customs concepts.

## 🎯 Learning Path Structure

### 12 Units (Topics)
1. **What Customs Is** - Foundation concepts
2. **Origins of Customs** - Historical beginnings
3. **Customs Across Continents** - Global development
4. **East African Customs Union** - Regional integration
5. **Global Customs Cooperation and WCO** - International cooperation
6. **Core Roles of Customs** - Practical functions
7. **Facilitation, Protection, and Smuggling Control** - Operational roles
8. **Customs Control and Barriers** - Control mechanisms
9. **Basic Control Tools and Customs Areas** - Practical tools
10. **Reports, Declarations, and Goods under Customs Control** - Procedures
11. **Legal Framework of Customs Work** - Legal foundations
12. **Smart Customs and Final Mastery** - Future and consolidation

### 6 Step Types per Unit
Each unit has 5-6 progressive steps:
1. 📚 **Learn** - Read and understand key concepts
2. 🧠 **Recall/Quiz** - Test your memory
3. 🎯 **Match** - Match terms with definitions
4. 💼 **Scenario** - Apply knowledge to real situations
5. ⏱️ **Timed Quiz** (optional) - Quick quiz under pressure
6. 👑 **Checkpoint/Boss Challenge** - Master the topic

## 🚀 Ideal User Journey

### Step 1: Landing & Overview
**What the user sees:**
- A visual map showing all 12 units in a path
- Unit 1 is unlocked and glowing
- Units 2-12 are locked (greyed out)
- User stats at top: XP (0), Streak (0), Hearts (5/5)
- Progress bar showing 0% complete

**User action:** Click on Unit 1 "What Customs Is"

---

### Step 2: Unit Overview
**What the user sees:**
- Unit title: "What Customs Is" 📚
- Description: "Understand the meaning of Customs..."
- 5 lesson circles in a vertical path
- Lesson 1 is unlocked, lessons 2-5 are locked
- Each lesson shows its type (Learn, Quiz, Match, etc.)

**User action:** Click on Lesson 1 (Learn type)

---

### Step 3: Learn Lesson (Lesson 1.1)
**What happens:**
1. **Educational Cards** (5 cards)
   - Card shows a question on front
   - User clicks to flip and see the answer
   - Example: "What is the trade-related role of Customs?" → "Facilitation of legitimate trade"
   - User clicks "Next" to see next card
   - Progress: 1/5, 2/5, 3/5, 4/5, 5/5

2. **Micro-Checks** (3 quick questions)
   - After cards, user gets 3 multiple-choice questions
   - Example: "Why did customs duties become a common source of public revenue?"
   - User selects answer
   - Immediate feedback: ✓ Correct (+10 XP) or ✗ Wrong (-1 heart)
   - Progress: 6/8, 7/8, 8/8

3. **Completion**
   - "Lesson Complete! +100 XP"
   - Confetti animation
   - Lesson 1 turns gold/completed
   - Lesson 2 unlocks
   - User returns to unit view

**User action:** Click on Lesson 2 (Quiz type)

---

### Step 4: Quiz Lesson (Lesson 1.2)
**What happens:**
1. **10 Multiple Choice Questions**
   - Question appears with 4 options
   - Example: "How does section 2 of EACCMA define Customs?"
   - User selects answer
   - Immediate feedback with explanation
   - Wrong answer = -1 heart
   - Progress bar: 1/10, 2/10... 10/10

2. **Pass Threshold: 60%**
   - User needs 6/10 correct to pass
   - If failed: "Try Again" button appears
   - If passed: "+100 XP, Lesson Complete!"

3. **Completion**
   - Lesson 2 turns gold
   - Lesson 3 unlocks
   - User returns to unit view

**User action:** Click on Lesson 3 (Match type)

---

### Step 5: Match Lesson (Lesson 1.3)
**What happens:**
1. **8 Matching Pairs**
   - Left column: Terms (e.g., "What is the WCO?")
   - Right column: Definitions (shuffled)
   - User drags or clicks to match
   - Correct match: ✓ Green highlight
   - Wrong match: ✗ Red shake, -1 heart
   - Progress: 1/8, 2/8... 8/8

2. **Pass Threshold: 70%**
   - User needs 6/8 correct matches
   - If failed: Retry with shuffled pairs
   - If passed: "+100 XP, Lesson Complete!"

3. **Completion**
   - Lesson 3 turns gold
   - Lesson 4 unlocks

**User action:** Click on Lesson 4 (Scenario type)

---

### Step 6: Scenario Lesson (Lesson 1.4)
**What happens:**
1. **6 Real-World Scenarios**
   - Scenario description appears
   - Example: "A shipment arrives without proper permits. What should Customs do?"
   - 4 options provided
   - User selects best answer
   - Detailed explanation shown
   - Wrong answer = -1 heart
   - Progress: 1/6, 2/6... 6/6

2. **Pass Threshold: 70%**
   - User needs 5/6 correct
   - If failed: Retry scenarios
   - If passed: "+100 XP, Lesson Complete!"

3. **Completion**
   - Lesson 4 turns gold
   - Lesson 5 (Checkpoint) unlocks

**User action:** Click on Lesson 5 (Checkpoint)

---

### Step 7: Checkpoint/Boss Challenge (Lesson 1.5)
**What happens:**
1. **12 Challenging Questions**
   - Comprehensive questions covering entire unit
   - Mix of concepts from all previous lessons
   - Higher difficulty than regular quizzes
   - Wrong answer = -1 heart
   - Progress: 1/12, 2/12... 12/12

2. **Pass Threshold: 75%**
   - User needs 9/12 correct to pass
   - This is the hardest lesson in the unit
   - If failed: Must retry (can review previous lessons first)
   - If passed: "+100 XP + 50 XP Mastery Bonus!"

3. **Unit Completion!**
   - 🎉 "Unit 1 Complete! +250 XP Chapter Bonus"
   - Unit 1 turns gold with crown icon
   - **Unit 2 unlocks!**
   - Progress bar updates: 8.3% complete (1/12 units)
   - User returns to main map

**User action:** Click on Unit 2 "Origins of Customs"

---

### Step 8: Progression Through Units 2-12
**The pattern repeats:**
- Each unit has 5-6 lessons
- Each lesson type teaches differently
- User must complete lessons in order
- Hearts regenerate over time or by practice
- XP accumulates (goal: 7,200 XP total)
- Streak counter increases with daily activity

**Key Mechanics:**
- **Hearts System**: Start with 5 hearts, lose 1 per wrong answer
- **XP System**: Earn 100 XP per lesson, bonuses for mastery
- **Streak System**: Daily activity maintains streak
- **Progressive Unlocking**: Must complete current lesson to unlock next
- **Retry System**: Can retry failed lessons anytime

---

### Step 9: Mid-Course Experience (Units 6-8)
**What's different:**
- User has built knowledge foundation
- Questions become more applied/practical
- Scenarios involve real customs procedures
- User sees progress: 50% complete, 3,600 XP
- Confidence builds with each completed unit

**Example Unit 7 Lesson:**
- "How Customs Facilitates Trade"
- Real questions about Revised Kyoto Convention
- Scenarios about AEO programs
- Match Incoterms with their meanings

---

### Step 10: Final Units (11-12)
**Unit 11: Legal Framework**
- Advanced legal concepts
- EACCMA schedules and regulations
- Complex scenarios about legal authority

**Unit 12: Smart Customs and Final Mastery**
- Future of customs administration
- Digital customs concepts
- **Final Boss Challenge**: Comprehensive test of entire course
- 12 questions covering all 12 units
- 75% pass threshold
- Highest difficulty

---

### Step 11: Course Completion! 🎉
**What happens:**
1. **Completion Screen**
   - "🎉 Congratulations! Course Complete!"
   - Final stats:
     - Total XP: 7,200+
     - Units Completed: 12/12
     - Lessons Completed: 67/67
     - Accuracy: XX%
     - Time Spent: XX hours
     - Streak: XX days

2. **Certificate/Badge**
   - "Tax and Customs Theory Master"
   - Completion date
   - Final score

3. **Options**
   - Review any lesson
   - Retake for higher score
   - Practice mode (all lessons unlocked)
   - Share achievement

## 📊 Gamification Elements

### XP (Experience Points)
- **Per Lesson**: 100 XP
- **Per Unit Completion**: 250 XP bonus
- **Perfect Boss Challenge**: +100 XP bonus
- **Unit Mastery**: +50 XP
- **Total Possible**: 7,200+ XP

### Hearts System
- **Starting Hearts**: 5
- **Lost per Wrong Answer**: 1
- **Recovery**: 
  - Wait timer (1 heart per hour)
  - Practice old lessons
  - Daily review bonus

### Streak System
- **Daily Activity**: +1 streak day
- **Streak Bonus**: Every 5 days = +20 XP
- **Streak Freeze**: 2 available (protect streak if you miss a day)

### Progress Tracking
- **Visual Map**: See all units and progress
- **Completion %**: Track overall progress
- **Mastery Levels**: 
  - Introduced (first attempt)
  - Practicing (retrying)
  - Strong (passed well)
  - Mastered (perfect score)

## 💡 Tips for Best Experience

### For Users:
1. **Complete lessons in order** - Each builds on previous knowledge
2. **Don't rush** - Take time to read explanations
3. **Review before checkpoints** - Go back to earlier lessons if needed
4. **Maintain your streak** - Daily practice reinforces learning
5. **Use hearts wisely** - Think before answering

### For Administrators:
1. **Monitor completion rates** - Track which lessons are difficult
2. **Review user feedback** - Adjust content difficulty
3. **Update content regularly** - Keep material current
4. **Encourage daily practice** - Promote streak system

---

## 🔄 Retry & Practice System

### If User Fails a Lesson:
1. "You need XX% to pass. You got XX%"
2. "Review the material and try again"
3. Option to review previous lessons
4. "Try Again" button available immediately
5. No penalty for retrying (except time)

### Practice Mode (After Completion):
- All lessons unlocked
- No hearts system
- Can replay any lesson
- Track improvement over time
- Useful for exam preparation
